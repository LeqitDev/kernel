CCOS

CCOS(CubeCoder['s]OperatingSystem) is a 32-Bit operating system for intel cpus.
On this page you can find information's about this project

Information's

This project started around the beginning of the year.
First I began to write the code in assembly but after a while I switched to C because C is easier to understand in contrast to assembly.
So first of all I created the main parts like text output for debugging, GDT/IDT and IRQs.
After a few lines of codes or even more I created a KeyboardController so I can write with the keyboard.
Then I programmed multitasking so the tasks can run side by side.
Because the tasks access the same memory I have to include paging.
The kernel can also load external program's as a task.
